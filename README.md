# 🌌 諸天萬界轉生系統 (Universal Rebirth System)
**Minecraft 1.20.1 | Forge 1.20.1**

這是一個將修仙與系統文元素帶入 Minecraft 的功能性模組。

### 🚀 核心機制
- **死亡覺醒 (Awakening)**: 玩家需經歷死亡以重組肉身並隨機抽取先天體質。
- **平滑曲線屬性 (Smooth Stats)**: 力量、敏捷、體質、感知、及四大元素皆為 1-100 級平滑成長。
- **連鎖挖礦 (Vein Miner)**: 土系屬性連動，等級越高連鎖數量越多 (最高 255)。
- **黑市商城 (Black Market)**: JSON 檔案驅動商品，支援花費代幣手動刷新。

### 📂 開發資訊
- 採用 **Client-Server Packet** 通訊架構，杜絕幽靈物品。
- 支援 **JSON 資料驅動**，無需改動程式碼即可自定義商城內容。
