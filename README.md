

# 🌌 諸天萬界轉生系統 (Universal Rebirth System)

> **「死亡不是終點，而是覺醒的開始。」 / "Death is not the end, but the beginning of awakening."**

《諸天萬界轉生系統》是一個為 Minecraft 帶來濃厚「修仙文 / 系統文」體驗的 RPG 模組。玩家將在經歷初次死亡後，綁定專屬系統、覺醒先天體質，並透過戰鬥與受傷獲取代幣，解鎖逆天改命的能力！

---

## 🔥 核心特色 / Core Features

### 💀 1. 破繭重生：先天體質系統 (Physique Awakening)

初次死亡不再是懲罰！玩家死後將啟動系統，並隨機抽取先天體質。每種體質將永久影響你的修煉之路：

* 🤍 **凡體 (Mortal Body)**: 平凡無奇，但潛力無限。
* 💛 **荒古聖體 (Ancient Holy Body)**: 煉體極難，但血量加成高達常人 3 倍！
* 🩵 **虛空劍心 (Void Sword Heart)**: 天生劍骨，力量與風系升級費用大幅減免。
* 💜 **混沌靈體 (Chaos Spirit)**: 元素寵兒，精通水火土三大元素法術。

### 📈 2. 登峰造極：1~100 級平滑屬性 (Smooth Leveling 1-100)

告別死板的技能樹！本模組採用平滑指數曲線，你的每一分代幣都能帶來真實的提升。

* ⚔️ **力量 (STR)**: 招招致命，刀刀附加真實傷害。
* 🛡️ **體質 (CON)**: 重塑肉身，大幅提升最大生命值。
* 👟 **敏捷 (DEX)**: 健步如飛，最高可達 +100% 常駐移動速度。
* 👁️ **感知 (PER)**: 滿級解鎖 30% 完美閃避，並具備半徑 50 格的生物透視掃描能力！

### 🌋 3. 掌控法則：四大元素能力 (Elemental Mastery)

* 🌪️ **風系 (WIND)**: 解鎖凌空踏虛，空中多段跳躍（滿級 5 段跳）。
* 🔥 **火系 (FIRE)**: 免疫高溫。80級攻擊附帶點燃，100級拳拳引發不破壞地形的猛烈爆炸！
* 💧 **水系 (WATER)**: 汪洋霸主。深海平滑回氧，50級以上解鎖水下超速挖掘。
* 🪨 **土系 (EARTH)**: 大地眷顧。除了物理減傷外，解鎖**「右鍵無損收割農作物＋雙倍掉落」**，並搭載 **BFS 演算法連鎖挖礦 (Vein Miner)**，滿級一鍵粉碎 255 格礦脈或參天大樹！

### 🏪 4. 萬界商城與命運抽籤 (Black Market & Gacha)

* 📦 **動態資料驅動 (JSON-Driven)**: 商城物品由外部 `examplemod_shop.json` 生成，伺服器主可隨時自定義商品與價格，免改程式碼！
* 🎲 **黑市刷新機制**: 每次開啟選單隨機進貨 4 件商品，玩家可花費 100 代幣手動洗牌，考驗血統與財力。
* 🎰 **命運寶箱**: 便宜的木寶箱有 80% 機率開出基礎物資；昂貴的黃金寶箱僅有 20% 機率，但一發入魂必出傳說神器！(獎勵池完全由伺服器端控制，杜絕客戶端作弊)。

## 💻 寫給開發者 / For Developers

本模組在架構設計上嚴格遵守 Forge 開發規範：

1. **Client-Server Packet Sync**: 完善的封包傳遞系統，UI (Client) 僅負責顯示與發送請求，扣款與物品發放絕對在 Server 端執行，防止「幽靈物品」與作弊。
2. **NBT Data Persistence**: 玩家屬性與金幣與 Player 的 PersistentData 深度綁定，支援跨維度傳送與死亡復活的資料繼承 (`PlayerEvent.Clone`)。
3. **BFS Vein Miner Algorithm**: 內建自製的廣度優先搜尋 (BFS) 連鎖挖礦演算法，確保遞迴尋路效能，並精準計算工具耐久度消耗。

---

### 🕹️ 如何遊玩 / How to Play

1. 安裝 Forge 1.20.1。
2. 將本模組 `.jar` 丟入 `mods` 資料夾。
3. 進入遊戲，勇敢地去「送死」一次以啟動系統。
4. 按下 **`J` 鍵** 打開科技感雙語 UI 介面，開始你的修仙轉生之旅！
